Debugging Lisp を翻訳しました

Dec 25th, 2015 3:26 am

Common Lispのデバッグに関する連載 Debugging Lisp の翻訳がひとまず終わりましたので、まとめてリンクを張っておきます。Lisp Advent Calendar 2015の第18日目として第1回の記事へのリンクを張ってましたが、こっちに張り替えます。

  • Debugging Lisp Part 1: 再コンパイル
  • Debugging Lisp Part 2: インスペクト
  • Debugging Lisp Part 3: クラスの再定義
  • Debugging Lisp Part 4: リスタート
  • Debugging Lisp Part 5: その他こまごま

つたない翻訳ではありますが、Common Lisp のデバッグ方法についてはあまり日本語で情報がなく、こういうのは必要なのではないかと思って訳してみた次第です。言語関係なく素晴らしい内容ですし。 著者の Michael Malis 氏のサイト http://malisper.me/ にはマクロについて他にも面白い記事があるので読める方は読むといいと思います。

もともとは Qiita のエントリーにするつもりだったんですが、どうも gif アニメーション画像に対応していないようだったので急遽 github io を使ってページを作ることにしました。テーマとかまだいじるかもしれません。
